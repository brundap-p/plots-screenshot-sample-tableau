Patient Data Analytics Dashboard (Tableau)

This project presents data analytics and visualization performed using Tableau on a patient dataset obtained for academic learning purposes.

The analysis focuses on exploring patient-related trends, patterns, and insights through interactive dashboards and visualizations. Various Tableau features, including charts, filters, and dashboards, were used to effectively represent the data and support data-driven interpretation.

Note: The original Tableau project files and exported reports were not retained. The available dashboard screenshots were recovered from archived copies, and therefore some image files may have WhatsApp-generated filenames (e.g., WhatsApp Image.jpg). These filenames do not affect the authenticity of the work and are used solely to preserve and showcase the project outcomes.

The screenshots included in this repository represent the original analysis and visualizations created as part of the project.

The original soft copy was unavailable; therefore, screenshots were captured from the available hard-copy documentation and uploaded for reference.
